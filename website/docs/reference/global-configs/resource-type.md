The `--resource-type` and `--exclude-resource-type` flags include or exclude resource types (such as unit tests) from the `dbt run`, `dbt build`, and `dbt clone` commands.

<File name='Usage'>

```text
$ dbt build --exclude-resource-type UNIT_TEST_NAME

```

</File>
