### Example query

The example query below outputs information about a metric. You can also add any field from the model endpoint; this example simply selects name. This includes schema, database, uniqueId, columns, and more. To learn more, refer to [Model object schema](/docs/dbt-cloud-apis/discovery-schema-model).


```graphql
{
  job(id: 123) {
    metric(uniqueId: "metric.jaffle_shop.new_customers") {
      uniqueId
      name
      packageName
      tags
      label
      runId
      description
      type
      sql
      timestamp
      timeGrains
      dimensions
      meta
      resourceType
      filters {
        field
        operator
        value
      }
      model {
        name
      }
    }
  }
}
```
