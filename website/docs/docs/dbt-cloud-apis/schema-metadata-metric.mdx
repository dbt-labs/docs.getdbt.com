### Example Queries
#### Metric information

The example query below outputs information about a metric.  Note that you can also add any field from the Model endpoint -- here we are simply selecting name. This includes schema, database, uniqueId, columns and more -- find documentation [here](/docs/dbt-cloud-apis/metadata-schema-model).


```graphql
{
  metric(jobId: 123, uniqueId: "metric.jaffle_shop.new_customers") {
    uniqueId
    name
    packageName
    tags
    label
    runId
    description
    type
    sql
    timestamp
    timeGrains
    dimensions
    meta
    resourceType
    filters {
       field
       operator
       value
     }
    model {
      name
    }
  }
}

```
